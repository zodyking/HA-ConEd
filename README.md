# HA ConEd Scraper

Home Assistant add-on repository for ConEd billing data scraper.

## Add-on

| Add-on | Description |
|--------|-------------|
| [ConEd Scraper](coned_scraper/) | Automated ConEd billing scraper with webhooks and web UI |

## Installation

1. Add this repository to your Home Assistant add-on store
2. Repository URL: `https://github.com/zodyking/HA-ConEd`
3. Install the **ConEd Scraper** addon
4. Start the addon and open the panel from the sidebar

## Documentation

See [coned_scraper/DOCS.md](coned_scraper/DOCS.md) for addon configuration and usage.
